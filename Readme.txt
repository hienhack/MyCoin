Individual Project - Mycoin

Running instruction

Server:
- Install packages: npm install
- Start server node: npm start
- Start mining node: node ./blockchain/miningNode.js

* The mining node must have started to be able to create transactions
* Use Keystore.txt to access initialized wallet with 100.000.000 coins

Web app:
- Install packages: npm install
- Start: npm run dev
